## 第一步：确定品牌标识及其目的

网站的跳动心脏就是品牌标识及其目的。

在这个例子中，我们使用一个英国搬家公司品牌 Anivan 作为示例。完全随机选择，没有任何隶属关系。但如果有的化，它们基本上是为您搬东西的公司。

我们想把这家公司打造成一个网站来销售它们。我已经销售了成千上万的网站，价值数千美元。我将向您展示的一些技巧会让您大吃一惊。

### 使用 Firecrawl.dev 提取品牌资产

首先，来到 **firecrol.dev** 进行品牌提取。在左边，点击 "Brand"（品牌）。这将从中获取所有品牌资产。

Firecrawl 最酷的地方在于他们有一个 API。因此，我们可以将其放入 API。所以我们实际上可以把它放到 Anti-Gravity 和 Claude Code 技能中。

基本上，如果我们想大规模地做到这一点，我们可以用这些信息进行物理操作。我们已经刮掉了品牌信息，可以看到我们拥有所有信息。

## 第二步：使用 Nano Banana 2 生成图像和动画

我们来到 Nano Banana 2，它非常酷，是最新、最强大的模型。我们想要生成一个 16x9 的图像。

我们需要做的是：

1. 获取一个起始图像
2. 获取该图像的解构版本
3. 获取将它们结合在一起的真实视频

### 使用 Claude Code 技能创建提示

首先，访问资源并下载"资产生成技能"。点击左侧并安装 Claude。一旦我们添加了技能，您就会看到它们显示在这里。

然后双击，召唤 Claude。使用技能创建者。如果您没有这个，您可以安装它，也可以直接与 Claude 交谈，告诉它使用 3D 资产生成器构建技能。

发电机在左侧。一旦完成，Claude Code 会继续为您测试。您可以完成整个流程并进行改进。它甚至带着三个不同的例子来到这里。

### 生成第一张图像

输入提示：
"嘿，使用这项技能，从一家搬家公司为我构建一个 HTML 视图，该公司充满了家具，我希望视觉效果是一辆满载物品的货车爆炸的后座。"

然后它将创建一个华丽的 HTML 界面供我们使用和复制提示。

复制组装好的提示，这是您想要的第一张图像。我们来到 Nano Banana 2，弹出这个。选择 "Little Hacks 16x9 和 1K"。

我们想要做的是获得货车标志。因此，我们前往 Google，抓住 Anivan 的标志。我们可以直接把标志放上去，这样它感觉更具体。

然后我们还想从网站上获取他们使用的实际货车。

### 生成解构版本

我们需要做的是获得任何货车标志。为此，我们来到 Nano Banana 2，选择 "Reference"（参考），引用之前的图像。然后粘贴这个版本。这样我们就引用了之前的图像，所以它将是那个版本的解构版本。

然后我们得到两个图像。我们可以将它们转换成视频。

### 创建视频

我们来到 Nano Banana 2 的视频功能，创建一个视频。选择开始帧（第一张）和结束帧。上传的同时，我们需要获取提示。

前往 Nano Banana 2 获取由 Claude Code 为我们创建的提示。复制它。来到 Hixelon，粘贴这个。不需要多镜头，不需要增强。关闭那些。Cling 3.0 就很好。7秒也可以。然后点击生成。

上传后，我们会得到一个视频。行李出来，家具飞出，前往不同的地方。来回播放以获得正确的过渡。做四五个不同版本，确保您对第一个和第二个非常满意。

## 第三步：使用 Claude Code 构建网站

下一步是使用 Anti-Gravity 中的 Claude Code 和我们的网站构建技能，将这些内容无缝地整合在一起，同时实际上利用设计元素和初始网站的跳动心脏。

### 下载必要的技能

前往社区，下载两个技能：
1. 资产生成（Asset Generation）
2. 3D 网站构建器（3D Website Builder）

将这些文件夹导入到 Anti-Gravity 中。

### 创建新项目

在 Anti-Gravity 中创建一个新文件夹。将项目命名为 "any van new website"。然后点击打开。

现在，我们激活 Claude Code。您会在左侧看到 Claude 扩展。如果您没有看到，可以在扩展中搜索并安装它。

### 使用滚动停止技能

我们将要使用的是滚动停止构建器技能。技能的妙处在于它会一步步引导我们完成所有操作。

它会问："您想使用哪个文件进行滚动驱动动画？" 我们选择"其他"并解释这是我们电脑上下载的最后一个文件。

这就是 Claude 的妙处。我们只需说："这就是它。请为我们做吧。"

提交答案后，它会一步步引导我们完成所有操作。

## 第四步：发布网站到 Vercel

现在我们已经创建了网站的 MVP。接下来我们想要：

1. 使其更加响应式
2. 将其托管到某个地方，使其拥有自己的域名
3. 可以发送给人们

### 连接 GitHub 和 Vercel

给 Anti-Gravity 这个提示：
"嘿，我想让您为我做两件事。我想让您连接到 GitHub，这样您就可以创建和发布仓库。第二件事是连接到 Vercel。"

GitHub 是我们放置所有信息的地方。Vercel 是实际为我们托管的地方。

### 创建 Vercel Token

前往 **vercel.com/account/settings/tokens**。创建一个 Token。给它一个名称，设置范围和过期时间。然后创建 Token。

然后回到 Anti-Gravity，向 Anti-Gravity 解释您的 API 密钥。访问 MCP 配置：前往 MCP 服务器，管理 MCP 服务，您会在左侧看到它。

点击 "View Raw Config"。然后您要说："嘿，我想让您将 Vercel MCP 与该 API 密钥一起添加。"

现在，一旦完成，您就能做一些非常棒的事情。当您点击刷新并自动出现时，您就知道它起作用了。

### 发布网站

现在您只需说：
"嘿，我想您去创建一个 GitHub 仓库。我想您发布它。然后我想您实际上在 Vercel 上为这个创建一个全新的网站。"

完成后，我们有两个链接：GitHub 的链接和网站的链接。

打开 GitHub 仓库，它已经在 GitHub 上线了。然后我们也有 Vercel 上网站的链接。

## 第五步：复制现有网站结构

这是一个非常有趣的技术。我向您展示 HTML 引用。

例如，如果我们喜欢某个网站的结构，我们可以：
1. 前往该网站
2. 前往 Google（如果我们将其发送给他们）
3. 搜索 "HTML website extractor"
4. 输入他们的网站
5. 下载为 HTML

然后回到 Anti-Gravity，打开一个新窗口，输入提示：
"嘿，我刚下载了原始网站的 HTML。我想您使用该 HTML 使用新副本和滚动停止动画重新创建它。"

关键是：您实际上要抓住他们的标志，抓住他们的排版，这样它看起来和感觉就像原始网站，我们只是添加了一些很棒的东西让它有一点pop。

### 利用现有脚手架

如果您想向现有企业推销，您可以利用他们现有的脚手架。

看，我们在新网站上添加了 Anivan 的标志。效果非常酷。而且看看它在移动端有多优化。

## 添加多页面和 SEO 优化

接下来我们要做的是能够在该网站上添加多个不同的页面，并进行 SEO 优化，这样我们就可以在 Google 上排名更高并获得更多客户。

### 创建页面

给 Anti-Gravity 这个提示：
"我有一个现有网站。看看我的当前网站，了解现有的设计语言。问我想要创建哪些页面。对于每个页面，匹配现有设计。添加完整的 SEO 优化。包括结构化数据。使其完全响应式，并更新所有页面的导航。创建所有页面后，对每个页面进行完整的 SEO 审计，给我一份报告。显示什么是优化的以及任何排名改进。"

我们使用 SEO 优化技能来做这件事。

### 导入 SEO 技能

前往 Claude Code，导入我们的 SEO 技能。如果您打开它，只需删除它，这样您就熟悉如何获取它。再次点击 Claude 东西。我可以输入 "/SEO"，我有我的 SEO 优化技能准备就绪。

如果您没有，只需前往技能创建者，然后您会经历整个过程。

### 创建多个页面

当 Anti-Gravity 实际构建不同的页面时，我们可以自己运行 SEO 优化技能。一旦您安装完毕，输入 "/SEO strategy"。然后它会问我们一个问题：您是否希望我对整个页面或只是文章进行 SEO 分析？

我们选择整个页面。然后我们可以给它我们要构建的任何 URL。我们给它一个新网站的 URL。发送它，让它为我们做完整的 SEO 策略。

### SEO 审计结果

Anti-Gravity 已经为我们创建了三个单独的页面：About 页面、Testimonials 页面和 Team 页面。

我们有了匹配的设计。我们有完整的 SEO、独特的标题、元描述、OG 标签、规范 URL、Twitter。我们甚至有 Twitter 卡片。JSON LD 结构化数据是响应式的。它有跨链接导航和共享 CSS 设计系统。

### SEO 审计报告

当 SEO 审计完成时，它会打开一个文件，显示您的分数。它会分解为什么好、为什么不。好，内容写得好，但正确使用了标题等。

给您一份执行摘要。给您一个页面分解。页面按页面显示发生了什么。全站问题、技术、内容策略等。

我喜欢的是内置的执行摘要，如果您前往关键词，也很酷。您有内部链接。所以就像所有页面链接到其他页面。

然后我们有一个行动计划，有所有它想做的事情。高影响力的东西、战略举措、完整的实施清单。
