## 关键要点整理

### 使用的工具

1. **Claude Code** - AI 编程助手，在类似 IDE 的环境中运行
2. **Nano Banana** - AI 图像生成工具，用于生成 3D 图像资源
3. **Kling 3.0** - 快手旗下 AI 视频生成模型，通过 Higsfield 平台访问

5. **Netlify** - 免费网站托管服务

| 项目 | 费用 |

| **总计** | **约 $2-3** |

### Kling 3.0 生成设置参数

- 模型：Kling 3.0
- 时长：5 秒
- 尺寸：16:9（横向）
- 分辨率：1080p
- 积分消耗：7.5 积分/个（约 $0.36）
- 背景：白色（根据需求）
- 质量：高

### 性能优化技巧

1. **帧提取优化**：将视频帧提取为优化的 JPEG 文件
2. **滚动关联**：将每个图像与滚动位置关联
3. **预加载**：实施预加载策略
4. **渐变遮罩**：使用向内渐变遮罩避免动画干扰文字
5. **文件压缩**：可将 5.3MB 压缩到 252KB

### 三步工作流

1. **编写 Prompt**：使用 Claude Code 配合技能库生成网站框架
2. **生成素材**：用 Kling 3.0 生成 3D 动画视频
3. **集成上线**：集成素材到网站并用 Netlify 免费托管
